# studious-spork

Studious Spork is an interactive coding tutor web app for learning Java, Python, JavaScript, and general programming concepts.

## Features

- Ask questions about Java, Python, JavaScript, and general coding topics
- See practical examples for common concepts
- Practice with training challenges and reveal solutions
- Local knowledge-base based tutor with a clean browser interface

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the tutor:

```bash
npm start
```

3. Open your browser at `http://localhost:3000`

## Project files

- `server.js` — Express server and API endpoints
- `knowledge.js` — built-in tutor knowledge base and training challenges
- `public/index.html` — frontend UI
- `public/script.js` — frontend interaction logic
- `public/styles.css` — user interface styling

## Learn more

Use the app to ask about classes, functions, loops, objects, and more. This starter project is also ready to extend with additional languages and AI integrations.
